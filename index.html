Guide Methodologique

1. Lecons : Selectionnez l'acte via les onglets, lisez les analyses scene par scene, puis cliquez sur "Evaluation" en bas de chaque fiche.

2. Vocabulaire Theatre : Consultez les 3 categories de vocabulaire, puis passez le QCM de chaque categorie.

3. QCM : Choisissez l'acte ou la categorie, repondez aux 10 questions. La correction s'affiche apres chaque reponse.

4. Moyenne : Consultez votre progression complete. Les QCM de Vocabulaire Theatre sont inclus dans la moyenne generale.
